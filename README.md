# About

- unclosable_window.exe

  它是一个无法关闭的窗口（虽然你可以用任务管理器）
  
- flying_mouse.exe

  它会让你的鼠标乱飞！
  
- Gemüse.exe

  它是一个小弹窗，每隔10秒会弹出 “Gemüse” 

- unclosable_window.vbs

  它是无法关闭的窗口的VBS版本（虽然你可以关闭wscript.exe）

- stupid.vbs

  它是一个弹窗，除非你承认 "You are stupid"，否则弹窗不会关闭！
